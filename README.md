# SimpleBili

基于 Flutter 开发的第三方 Bilibili跨平台客户端。避免大量的推荐视频、广告和复杂的页面，该项目推出了更加简化版的B站，让大家专注于自己想要看的视频和关注的up，而不是被推荐算法偷走了注意力。



## 🌟 项目特性 / Features

本项目目前实现了以下核心功能模块：

*   **🔑 认证授权 (Auth):**
    *   登录页面支持 (Web 扫码或cookie登录)
*   **📺 首页动态 (Feed):**
    *   获取视频推荐与动态内容加载
    *   视频卡片与动态图文卡片组件展现
*   **▶️ 视频播放 (Player):**
    *   视频解析与播放功能
*   **🔍 搜索功能 (Search):**
    *   关键词搜索视频及综合结果
*   **👤 UP主页 (UP):**
    *   查看 UP 主投稿视频

## 🚀 快速开始 / Getting Started

### 环境要求
- Flutter SDK (推荐使用最新 Stable 版本)
- Dart SDK

### 运行项目

1. 克隆项目到本地:
   ```bash
   git clone <你的项目地址>
   cd SimpleBi
   ```

2. 获取依赖:
   ```bash
   flutter pub get
   ```

3. 运行项目 (连接设备或启动模拟器):
   ```bash
   flutter run
   ```
